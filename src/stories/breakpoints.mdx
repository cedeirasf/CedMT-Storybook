# Breakpoints

Los Breakpoints definen cómo las interfaces se adaptan a diferentes tamaños de pantalla. TailwindCSS ofrece clases predefinidas para los breakpoints más comunes.

## Ejemplo de uso

```tsx
<div className="hidden md:block">
  Este texto es visible desde pantallas medianas (md) hacia arriba.
</div>
<div className="hidden lg:block">
  Este texto es visible desde pantallas grandes (lg) hacia arriba.
</div>
